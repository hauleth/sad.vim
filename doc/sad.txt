INTRODUCTION                               *sad*

This plugin provides several handy shortcuts for quick search and replace.

SEARCH                                     *sad-search*

                                           *v_star*
*                   Search forward for visual selection

                                           *v_#*
#                   Search backward for visual selection

REPLACE                                    *sad-replace*


                                           *s*
s{motion}           Delete {motion} text and start insert. Afterwards |.| can
                    be used to replace next occurence of deleted text.

                                           *ss*
{count}ss           Delete {count} lines downwards |linewise|. After that |.|
                    will change next occurence of that lines.

                                           *S*
S{motion}           Same as |s| but then replaces previous occurence.

                                           *SS*
{count}SS           Delete {count} lines downwards |linewise|. After that |.|
                    will change previous occurence of that lines.

                                           *v_s*
{Visual}s           Delete highlighted text, set search to it and start insert
                    (for {Visual} see |Visual-mode|). Afterwards |.| can be used
                    to replace next occurences of selected text.

                                           *v_S*
{Visual}S           Same as |v_s| but then replaces previous occurence.
